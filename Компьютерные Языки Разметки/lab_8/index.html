Стиль, которым в итоге будет оформлен элемент, состоит из набора пар свойств и их значений. К одному элементу может применяться сразу несколько таких пар. Например, текст может быть по цвету красным, по размеру 25 пикселей и оформляться полужирным шрифтом.

Свойство - это характеристика, которая будет применяться к элементу (в нашем примере это цвет). Значение - это конкретное значение выбранного свойства (в нашем примере это красный). Свойство CSS отделяется от его значения знаком двоеточия и после указания значения ставиться знак точка с запятой. Это делается для того, чтобы отделить одно от другого.
